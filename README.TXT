PROJECT TITLE: Flappy Bird Game
PURPOSE OF PROJECT: The purpose of this project is to recreate the popular Flappy Bird mobile game for computers, providing an entertaining and challenging gaming experience.

HOW TO START THIS PROJECT: 
Step 1: Download GreenFoot (https://www.greenfoot.org/download) or any application that supports JavaScript. 
Step 2: Open the file and the game will load
Step 3: To play click start and use the spacebar to make the bird flap its wings and navigate through the pipes. 
Step 4: Have Fun!
